function dnaStrand(dna){
  let object = {A:'T', T:'A', C:'G', G:'C'}
  return dna.replace(/./g, e => object[e])
}

/*
Комментарий: Эта функция dnaStrand принимает строку dna в качестве аргумента и заменяет каждый символ в строке на соответствующий символ ДНК.

Объяснение кода:

Создается объект object, который содержит соответствия для символов ДНК. Ключами в этом объекте являются символы ДНК (A, T, C, G), а значениями - символы, которые должны заменить соответствующие символы ДНК.
Возвращается результат вызова метода replace на строке dna.
Метод replace принимает два аргумента: регулярное выражение /./g и функцию обратного вызова e => object[e].
Регулярное выражение /./g соответствует любому символу в строке.
Функция обратного вызова e => object[e] принимает текущий символ e и возвращает соответствующий символ из объекта object.
Таким образом, каждый символ в строке dna будет заменен на соответствующий символ ДНК, используя объект object.
Итоговая строка с замененными символами ДНК возвращается в качестве результата.


*/
